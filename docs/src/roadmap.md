# Roadmap

Crucible is moving from an MVP review loop toward a broader local review platform.

## Current themes

- richer context gathering
- stronger convergence and consensus logic
- deeper deterministic prechecks
- better session persistence and resumability
- tighter GitHub review publishing

## Planned expansions

- full multi-agent council behavior
- daemon mode for always-on local review
- temporal review memory
- adversarial validation of generated auto-fixes
- confidence weighting based on historical reviewer accuracy

The detailed planning material remains in `docs/specs/roadmap.md`.
